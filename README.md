# easy-123panfx

Tampermonkey / 油猴脚本，简化 [123panfx.com](https://www.123panfx.com) 网盘链接获取流程。

## 适用网站

- https://www.123panfx.com/?* （所有带查询参数的页面）

## 用途

在 123panfx.com 上获取资源链接时，通常需要先回帖才能看到隐藏内容。本脚本自动化这个流程：

- **按下 `Ctrl+C`** → 自动填写回帖内容并提交
- **回帖成功后** → 自动在弹出的成功提示中提取链接并打开

## 用法

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击扩展图标 → 添加新脚本 → 将 `easy-123panfx.js` 内容粘贴进去保存
3. 打开任意 123panfx 资源页，看到网盘链接时按 `Ctrl+C` 即可
